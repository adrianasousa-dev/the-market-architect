📊 The Market Architect v13.1

O The Market Architect é um motor avançado de Big Data e Inteligência Artificial desenvolvido para consolidar ecossistemas de dados massivos, processando mais de 390.000 logs para a extração estratégica de personas e mecanismos de conversão.

🧠 Visão Geral

Este projeto representa a camada de inteligência analítica da minha stack tecnológica. Foi desenhado para resolver o problema do "ruído" em grandes volumes de dados, utilizando Python (Pandas) para a estruturação e a API do Gemini para a interpretação qualitativa e geração de insights de mercado.

🛠️ Funcionalidades de Elite

Processamento de Escala: Arquitetura otimizada para ingestão e limpeza de bases com +390k registos.

AI Insight Engine: Integração nativa com LLMs para transformar dados brutos em inteligência acionável.

Mapeamento de Avatares (ICP): Identificação automática do Perfil de Cliente Ideal baseado em padrões comportamentais detetados nos logs.

Segurança de Dados: Implementação rigorosa de variáveis de ambiente para proteção de credenciais sensíveis.

💻 Stack Técnica

Linguagem: Python 3.11+

Análise de Dados: Pandas, NumPy.

Inteligência Artificial: Google Generative AI (Gemini 1.5 Flash).

Ambiente: Gestão de dependências via requirements.txt e segurança via .gitignore.

📂 Estrutura do Repositório

main_architect.py: O núcleo de processamento e inteligência.

requirements.txt: Lista de bibliotecas necessárias para execução.

.gitignore: Protocolo de segurança para proteção de chaves de API.

.env.example: Modelo para configuração de variáveis de ambiente locais.

🚀 Como Configurar e Executar

Clone este repositório para a sua máquina local:

git clone [https://github.com/adrianasousa-dev/the-market-architect.git](https://github.com/adrianasousa-dev/the-market-architect.git)


Instale as dependências necessárias:

pip install -r requirements.txt


Crie um ficheiro .env na raiz do projeto e adicione a sua chave:

GEMINI_API_KEY=sua_chave_aqui


Execute o motor principal:

python main_architect.py
